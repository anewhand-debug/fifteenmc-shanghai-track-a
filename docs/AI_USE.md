# AI Assistance Statement

This project used AI assistance to scaffold code, organize notebook structure, draft reusable helper functions, and prepare documentation templates.

The human author remains responsible for:

- Selecting and validating data sources.
- Running notebooks end to end.
- Checking source licensing and platform terms of service.
- Interpreting results and making design decisions.
- Revising weights, assumptions, and limitations where course feedback requires it.

No API keys or private credentials were provided to or embedded by the AI assistant. Any sample files in `data/sample/` are explicitly labeled as samples and must not be represented as real observed data.
